Who has been to a Code Dojo before? The idea is that it's a place to practice programming without the pressures of work/school. It doesn't matter if you don't complete the problem - as long you learn and have fun. Also, it gives you the oppotintity to try out new ideas, languages or working with different people. Like driving a car - you might drive 10 miles to work every day, but if it's the same route you won't really be improving.

Who has done an Advent of code problem before? Who has done any this year? There is a different problem each day. Each problem is in two parts, successfully completing the first part unlocks the second part. They generally get harder as the month goes on, sometimes the weekend puzzles are more involved than the weekdays. Each also has a story line based around the puzzles. Different people do the puzzles for different reasons * Practice puzzles * Solve the puzzle the fastest and get on the leader board (very heard) * Learn a new language. * Write a fast solution
